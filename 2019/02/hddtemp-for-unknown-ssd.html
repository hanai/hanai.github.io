使用 hddtemp 于未知 SSD

发表于 2019-02-16 更新于 2022-04-24 本文字数： 3.8k 阅读时长 ≈ 7 分钟

部分 SSD 无法被 hddtemp 获取到温度信息：

$ sudo hddtemp /dev/sda
WARNING: Drive /dev/sda doesn't seem to have a temperature sensor.
WARNING: This doesn't mean it hasn't got one.
WARNING: If you are sure it has one, please contact me (hddtemp@guzu.net).
WARNING: See --help, --debug and --drivebase options.
/dev/sda: Samsung SSD 860 EVO 500G B              @:  no sensor

安装 smartmontools：

$ sudo apt-get install smartmontools

使用 smartctl 查看设备的 SMART 信息：

可以找到温度传感器对应的 ID 为 190。

找到 hddtemp.db 文件：

$ sudo find -name 'hddtemp.db'
./etc/hddtemp.db

编辑文件，增加相关设备的信息：

"Samsung SSD 860 EVO 500G B"    190     C "Samsung SSD 860 EVO 500GB"
